# Face Recognition Web Application

A real-time AI-powered face detection and recognition system built with Flask, OpenCV, and the face_recognition library. This application provides a modern, responsive web interface for live face recognition through your webcam.

![Python](https://img.shields.io/badge/Python-3.12-blue?logo=python)
![Flask](https://img.shields.io/badge/Flask-3.1-green?logo=flask)
![OpenCV](https://img.shields.io/badge/OpenCV-4.13-red?logo=opencv)

## Features

- **Real-time Video Streaming** - Live webcam feed with face detection
- **Face Recognition** - Identify known faces in real-time
- **Modern UI** - Dark theme with animated particles and glowing effects
- **Responsive Design** - Works on desktop and mobile devices
- **Live Indicator** - Visual feedback showing camera status
- **Fast Processing** - Optimized for real-time performance
